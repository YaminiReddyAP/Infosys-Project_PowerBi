Abstract:


This project involves the analysis of Indian General Election data using Power BI with the aim of presenting election information in a visual and interactive manner. It includes analysis of election results across multiple years, states, political parties, and candidates, along with demographic insights of election winners. Power BI Desktop is used to build dashboards, Power Query is applied for data transformation, DAX is used to calculate key metrics, and Excel/CSV files are used as source data. The project helps in understanding voting behavior, party performance, and overall election trends through data visualization.
